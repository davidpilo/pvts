Example: Fibonacci Calls It'll blow your mind.

The figure on the right is an example of running the code below in the interpreter that demonstrates how the Fibonacci Algorithm works in a functional programming enviroment.

; Lets define Fibonacci:
(define (fib n)
(cond
((= n 1) 1)
((= n 0) 0)
(else (+ (fib (- n 1)) (fib (- n 2))))))

; (fib 6) evaluates to
(fib 6)
